AllAI Clipboard – Local Setup Guide

AllAI Clipboard is a browser extension that lets you save and reuse prompts across major AI platforms. Instead of rewriting the same prompts, you can store them once and quickly insert them into any AI chat box.

Features
Save custom prompts for reuse
Quick access from your browser toolbar
One-click prompt insertion into AI chat boxes
Works across multiple AI platforms
How It Works
Pin the extension to your browser
Click the extension icon to open your saved prompts
Select a prompt
The prompt is automatically inserted into the AI chat box
Click send and use it instantly
How to Load the Extension Locally
Step 1: Download or Clone the Repository
Download the ZIP from this repo and extract it
or
Clone the repository:
git clone <your-repo-url>
Step 2: Open Extensions Page

For Chrome:

Open Chrome
Go to chrome://extensions/

For Edge:

Open Edge
Go to edge://extensions/
Step 3: Enable Developer Mode
Turn on Developer mode (top right corner)
Step 4: Load the Extension
Click Load unpacked
Select the folder containing the extension files, make sure it includes manifest.json
Step 5: Pin the Extension
Click the extensions icon in your browser
Find AllAI Clipboard
Click the pin icon so it appears in your toolbar
Step 6: Start Using It
Click the extension icon
Add or select a saved prompt
The prompt will automatically appear in your AI chat box
Press send to use it
Notes
Keep the folder structure unchanged
Reload the extension from the extensions page after making code changes
Troubleshooting
If the extension does not load, check for errors on the extensions page
Ensure manifest.json is valid and in the root folder
Make sure the extension has permission to interact with the sites you are using
